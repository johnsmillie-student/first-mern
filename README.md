## Packages

### Backend:
- Express.js
- MongoDB
- Mongoose

```
$ cd backend
  npm install
  npm start
```
**API Endpoints @**: http://localhost:4000

### Frontend:
- Vite React.js
- Materialize
- Axios

``` 
$ cd frontend
  npm install
  npm run dev
```
**Open Browser:** http://localhost:5173
